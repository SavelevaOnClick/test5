1. методы-глаголы REST API - CRUD.
Это create использоуется для создание новой записи(POST запрос относится)
Read используется для чтения получения данные с сервера(GET апрос относится)
Update используется для обновления определённых данных(относится PUT и PATCH)
-PUT полностью заменяет данные определённой записи по id
-PATCH для частичной замены какого либо поля и добавления какого либо поля
Delete спользуется для удаленния данных (DELETE запрос)

2.Литерал используются для обьявления определённых данных, тоесть литерал это явно определённые данные
Литералы бывают строковыми, логическими, литералом массива,
 литералами обьекта, литералом целого числа, литерал числа с плавающей точкой и литералом регулярного
выражения
пример литерала массива 
const animals = ['cat', 'dog'];
литерал обьекта
const user = {
  name: 'Vasya',
  age: 66
} 
литерал регулярного выражение
const regExp = /\w*/
пример строкового литерала
const message = 'hello, word!'
пример целочисленного литерала
const age = 55
